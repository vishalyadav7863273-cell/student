Features · विशेषताएं

क्या-क्या मिलेगा?

Real EVM + Persistent Database — सब एक ही file में।

💾
IndexedDB Database
Browser की built-in IndexedDB में data permanently store होता है — page refresh के बाद भी data safe रहता है।
⚡
Auto-Save Every Vote
हर vote cast होने के बाद तुरंत database update होता है — कोई data loss नहीं।
📁
Session History
सभी past voting sessions save हैं। Click करके details देखें, delete करें।
👥
Voter Queue
Registered voters की live queue — active, waiting, voted सब clearly दिखता है।
❌
NOTA Option
None of the Above — real Indian EVM की तरह NOTA भी available है।
🔊
Audio + Animations
Vote confirm पर beep sound और smooth page transitions।
